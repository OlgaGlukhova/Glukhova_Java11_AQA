Задание 4

Данную последовательность скобок  [((())()(())]] нельзя считать правильной, потому что она не удовлетворяет правилу "Правильной скобочной последовательности (ПСП)".
Необходимо изменить следующее: для каждой вложенной подпоследованности прописать открытую и закрытую одного типа
 [ [ ( ( () ) () ( () ) ) ] ]
